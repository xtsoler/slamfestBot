Available commands:
Command Available to Description
!g or !guess Everyone Select one of the available choices during a slamfest round e.g. !g c1
!available Everyone The bot will reply with a list of all the currently available choices that are not taken by another user.
!taken Everyone The bot will show all the already taken choices of the current round and also display which user has each of these choices.
!points Everyone Shows how many wins and how many points you currently have. Also shows what choices you had on your wins. You can specify a user to see their stats e.g. !points user_x
!standings Everyone Current standings of the all the users that have at least one point will be shown.
!show Everyone !show item_type e.g. !show amulet will show all the possible choices for the specified item type. If no item type is specified, the choices for the current item type of the ongoing round will be displayed.
!slam_log Everyone A list of all the completed rounds that had a winner will be displayed.
!end_round Streamer and mods Forced end of current round is performed by the bot. This is to stop an ongoing round that was started by mistake with the wrong item type for exmaple.
!go_away Streamer and mods The bot will leave the chatroom.
!reset_standings Streamer and mods Current standings will be deleted. All users will lose their recorded wins.
!add_winner Streamer and mods Manually add a winner with the details of this virtual round. For example !add_winner amulet user_x c1 c1 means that user_x won an amulet round and winning choice was c1 and bonus choice was also c1. This will help to correct errors of previous rounds.
!delete_last_winner Streamer and mods This command will delete the last winner entry of the winners recorded by the bot.
!finish Streamer and mods !finish c1 will end the current round with the winning choice of c1. The winner will be appointed the calculated points and an entry will be recorded on the slam log.
!bonus Streamer and mods Specify a bonus choice manually with !bonus r1 for example. The bonus will be set automatically with !prep command but it can be changed manually with this command. If a round is ongoing !bonus with no specified option will print the current bonus choice.
!prep Streamer and mods !prep amulet will prepare a round for the item type amulet. If the item type specified is helm, armor, weapon or shield, the bot will require you to specify the number of possible sockets. !prep armor 2 will prepare an armor and add up to 2 sockets as choices. !prep armor 0 will prepare a round of armor with no socket choices available.
!start Streamer and mods Start the round and accept guesses by the users.
!block Streamer and mods Stop accepting guesses from the users.
!allow Streamer and mods Allow guesses from the users again for the currently ongoing round.
!accept_blocked Streamer and mods If anyone tried to guess while guesses were blocked, this command will accept that guess if it is still valid to do so.
!verbose Streamer and mods The bot will type the available options in the chat.
!laconic Streamer and mods The bot will not type the available options in the chat, they will only be shown on the status page.
!stats Everyone The bot will type some stats in the chat, e.g. total slams, total bricks.
